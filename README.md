# MEC: Medical Evidence Capsules Dataset

This repository provides the dataset used in the MICCAI 2026 paper:

**MEC: Medical Evidence Capsules for Retrieval-Augmented Generation in Medical Multimodal Question Answering**

---

## 📌 Dataset Availability

Due to the large scale of the dataset, the full version is hosted on Hugging Face Datasets:

https://huggingface.co/datasets/Morefreedai/MEC

---

## 📁 Dataset Description

The MEC dataset is designed for medical retrieval-augmented generation (RAG) and medical multimodal question answering tasks. It organizes heterogeneous medical evidence into structured formats to support reliable medical reasoning.

---

## 📚 Data Sources

The dataset is constructed from multiple publicly available medical resources:

- Textual QA data: derived from MedQA  
- Knowledge graph / medical ontology: derived from UMLS (Unified Medical Language System)  
- Medical imaging-text pairs: derived from MIMIC-CXR
